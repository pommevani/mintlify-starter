## Whitelist
**WHY IS NUGGETS LEAVING? WHY IS HE SENDING A MESSAGE?**

<Info>
    To make sure we know the server owners, you need to be whitelisted. 
    But how to make it? To keep it simple, read this:
</Info>

<Steps>
  <Step title="Join our server in the navigation bar (Support)" description="You will be able to use this server for any bug reports or help.">
    Click on the Support link in the navigation bar to join our Discord server.
  </Step>

  <Step title="Open a ticket to get support" description="You will have to open a ticket, this is for advanced help when the query is complex.">
    Go to the ticket channel and create a new support ticket.
  </Step>

  <Step title="Send your request and why do you need flare" description="Any reason, it can be long or not but if possible precis for what features you are interested with.">
    Explain your use case and which features interest you the most.
  </Step>

  <Step title="Get whitelisted and enjoy our app." description="We hope you will enjoy our app and it's features :)">
    Once approved, you'll get full access to use flare in your server.
  </Step>
</Steps>
